This is a Vortex Lattice Method code with an Evolutionary Optimisation algorithm (Modified Cuckoo Search) wrap-around implemented in MATLAB.

Developed by Dr Sean Walton, Dr Ben Evans, Chris Walton and Artemi Sattarov at Swansea University, UK.

Initial geometry is input via a .xls (Excel) file.

Main programme is main.m

Further details available in the User_Guide.pdf



REFERENCES:

Walton, Sean & Hassan, O & Morgan, K & Brown, M. (2011). Modified cuckoo search: a new gradient free optimisation algorithm. Chaos, Solitons and Fractals, 44, 710-718. Chaos Solitons & Fractals. 44. 710-718. 10.1016/j.chaos.2011.06.004.
